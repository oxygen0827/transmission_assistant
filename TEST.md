# 功能验证手册

按以下步骤逐项测试，确认每个功能是否正常。

---

## 准备工作

1. 创建 `.env` 文件，填入 `GLM_API_KEY`
2. 安装依赖：`pip install -r requirements.txt`
3. 启动后端：`uvicorn backend.main:app --reload --port 8000`
4. 启动前端：`cd frontend && npm run dev`
5. 打开 `http://localhost:5173`
